Abrir Poor Maps

Abre Poor Maps y accede a las herramientas pulsando el botón de la parte inferior de la pantalla (resaltado con el recuadro rojo en la captura de pantalla).

Desplaza hacia arriba para continuar

Mapas de referencia

Para configurar el dibujo de mapas, haz clic en "Basemaps", como se muestra.

Desplaza hacia arriba para continuar

Selecciona OSM Scout Server entre la lista de mapas de referencia. Se recomienda usar @2x para un renderizado más rápido.

Desplaza hacia arriba para continuar

Buscar

En la selección de herramientas de Poor Maps, selecciona "Search", como se muestra.

Desplaza hacia arriba para continuar

En Search, elige OSM Scout Server como proveedor del motor de búsqueda, como se muestra.

Desplaza hacia arriba para continuar

Navegación

En la selección de herramientas de Poor Maps, selecciona "Navigation", como se muestra.

Desplaza hacia arriba para continuar

En Navigation, elige OSM Scout Server como proveedor del motor de búsqueda, como se muestra.

Desplaza hacia arriba para continuar
