Abrir modRana

Abre modRana y accede a las herramientas pulsando el botón de la parte inferior izquierda de la pantalla (resaltado con el recuadro rojo en la captura de pantalla).

Desplaza hacia arriba para continuar

Capa de mapa

Para configurar el dibujo de mapas, haz clic en "Map"

Desplaza hacia arriba para continuar

Accede a "Main map".

Desplaza hacia arriba para continuar

Selecciona OSM Scout Server entre la lista de capas de mapas.

Desplaza hacia arriba para continuar

Opciones

Después de seleccionar la capa de mapa, desliza hacia la derecha hasta llegar a la página principal. Selecciona "Options".

Desplaza hacia arriba para continuar

POI

En Options, selecciona POI.

Desplaza hacia arriba para continuar

POI

Selecciona OSM Scout Server como proveedor del motor de búsqueda de lugares y búsqueda local, como se muestra.

Desplaza hacia arriba para continuar

Navegación

Después de configurar de la búsqueda, desliza hacia la derecha hasta llegar a Options. En Options, selecciona Navigation.

Desplaza hacia arriba para continuar

Navegación

En los ajustes de navegación, selecciona OSM Scout Server como proveedor del motor de cálculo de rutas, como se muestra.

Desplaza hacia arriba para continuar

Funcionamiento

¡Ya está listo! No olvides que OSM Scout Server y modRana tienen que estar ejecutándose simultáneamente para que funcionen los mapas sin conexión.
