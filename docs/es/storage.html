Antes de asignar la carpeta de almacenamiento, se mostrará un mensaje solicitando que lo hagas. Este tutorial te mostrará cómo asignar una carpeta de la tarjeta SD u otra ubicación no predeterminada.

Desplaza hacia arriba para continuar

Abrir FileCase

Abre una aplicación de explorador de archivos en tu dispositivo Sailfish. Por ejemplo, abre Filecase y desliza hacia la izquierda, como muestra la captura de pantalla.

Desplaza hacia arriba para continuar

Tarjeta SD

Haz clic en Tarjeta SD, como se muestra.

Desplaza hacia arriba para continuar

Crear nueva carpeta

Navega dentro de la tarjeta SD hasta la ubicación donde quieres crear tu carpeta, y créala desde el menú deslizante, como se muestra.

Desplaza hacia arriba para continuar

Nombre de la nueva carpeta

Nombra la nueva carpeta como desees. En este ejemplo, el nombre usado es Maps.OSM.

Desplaza hacia arriba para continuar

Volver a OSM Scout Server

Vuelve a OSM Scout Server y abre los Ajustes desde el menú deslizante, como se muestra.

Desplaza hacia arriba para continuar

Selección del almacenamiento

Abre el diálogo de selección del almacenamiento, como muestra el elemento resaltado de la captura de pantalla.

Desplaza hacia arriba para continuar

Navegar hasta la carpeta asignada

Navega hasta la ubicación de la carpeta que vas a asignar. En este ejemplo, partimos de /home/nemo/Documents/ y tenemos que llegar a /media/sdcard/... . Por tanto, primero desplázate hacia arriba y presiona .., como se muestra.

Desplaza hacia arriba para continuar

Cuando llegues al directorio raíz (/), busca la carpeta media.

Desplaza hacia arriba para continuar

Selecciona la carpeta media, como se muestra, y navega dentro hasta encontrar la carpeta de la Tarjeta SD que deseas asignar.

Desplaza hacia arriba para continuar

Seleccionar la carpeta asignada

Cuando veas la carpeta que quieres asignar, haz clic en ella y mantén pulsado, hasta que se abra el menú contextual. Presiona Seleccionar para confirmar tu selección.

Desplaza hacia arriba para continuar

Has asignado la nueva carpeta, como muestra la captura de pantalla en "Almacenamiento de mapas". Continua con Aceptar para confirmar los nuevos ajustes.
