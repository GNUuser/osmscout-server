Before storage is allocated and setup, you will be greeted with the message asking you to do so. This tutorial will show how to allocate storage on SD Card or other non-default location.

Create new folder

When you navigate on SD Card into location where you want your new folder, create it by opening pulley menu, as shown

Name new folder

Name new folder as you wish. In this example, Maps.OSM is used.

Back to OSM Scout Server

Go back to OSM Scout Server and open its Settings by opening pulley menu, as shown.

Storage selection

Open the storage selection dialog as shown by the highlighted element on the screenshot.

Navigate to allocated folder

Navigate to the location where the allocated folder is. In this example, we start from /home/nemo and have to get to /media/sdcard/... . So, navigation starts with moving up by pressing .. as shown.

Scroll down for continuation

When arrived to root (/), look for media folder.

Scroll down for continuation

Select media folder, as shown, and move along the file tree to the allocated folder on SD Card until you see it.

Scroll down for continuation

Select allocated folder

When you can see the allocated folder, click on it and hold, until you see opening of the context menu. Press Select to confirm your selection.

Scroll down for continuation

The new allocated folder is selected, as shown in the screenshot under "Maps storage". Proceed with the Accept to confirm your new settings.
