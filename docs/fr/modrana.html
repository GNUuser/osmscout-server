Ouvrir modRana

Ouvrir modRana et accéder au menu en appuyant sur le bouton en bas à gauche de l'écran (tel qu'indiqué par l'encadré rouge sur la capture d'écran).

Défiler vers le bas pour continuer

Fond de cartes

Pour configurer l'affichage des cartes, cliquer sur "Map".

Défiler vers le bas pour continuer

Sélectionner "Main map".

Défiler vers le bas pour continuer

Sélectionner OSM Scout Server parmis la liste des cartes.

Défiler vers le bas pour continuer

Options

Après avoir sélectionné le fond de carte, glisser vers la droite pour retourner au menu. Sélectionner "Options".

POI

Dans les Options, sélectionner POI

POI

Sélectionner OSM Scout Server comme fournisseur de recherche pour les adresses et points d'intérêts, comme indiqué.

Navigation

Après la configuration de la rechercher, revenir au menu d'options, puis sélectionner Navigation.

Navigation

Dans les paramètres de navigation, sélectionner OSM Scout Server comme fournisseur de directions, comme indiqué.

Défiler vers le bas pour continuer

Mode of operation

La configuration est terminée ! N'oubliez pas que vous devez avoir OSM Scout et modRana lancés simultanément pour que le mode hors-ligne fonctionne.
