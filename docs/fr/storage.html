Avant que le stockage soit alloué, vous allez être acceuillis avec un message vous demandant de la faire. Ce tutoriel va vous présenter comment allouer le stockage sur la carte SD ou tout autre emplacement différent de l'emplacmeent par défaut.

Create new folder

Lorsque vous avez navigué jusqu'à l'emplacement où vous souhaitez créer votre nouveau dossier, créez le tel qu'indiqué sur la capture d'écran.

Nommer le nouveau dossier

Nommer le nouveau dossier comme vous le souhaitez. Dans cet exemple, Maps.OSM est utilisé.

Retour au serveur OSM Scout

Retournez au serveur OSM Scout et entrez dans les paramètres en ouvrant le menu descendant, comme illustré.

Sélection de l'emplacement de stockage

Ouvrir la page de sélection de l'emplacement de stockage tel qu'indiqué par l'élément en surbrillance dans la capture d'écran.

Naviguez au dossier alloué

Naviguez à l'emplacement où le dossier alloué est. Dans cet exemple, nous partons de /home/nemo et devons aller à /media/sdcard/... . Donc, la navigation commence par remonter en appuyant sur .. comme illustré.

Lorsqu'arrivé à la racine (/), recherchez le dossier media.

Défiler vers le bas pour continuer

Sélectionner le dossier media, puis remontez le système de fichier jusqu'à atteindre le dossier alloué sur la carte SD.

Défiler vers le bas pour continuer

Selectionner le dossier alloué

Lorsque vous voyez le dossier alloué, maintenir appuyé dessus, jusqu'à apercevoir le menu contextuel. Appuyer sur Sélectionner pour confirmer la sélection.

Défiler vers le bas pour continuer

Le nouveau dossier allouée est alors sélectionné, comme indiqué par la capture d'écran sous "Stockage des cartes".
