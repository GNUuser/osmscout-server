Starta modRana

Starta modRana och gå till verktygsmenyn genom att trycka på knappen i nedre vänstra hörnet på skärmen (markerat med röd fyrkant på skärmklippet).

Skrolla neråt för att fortsätta

Kartlager

Tryck på "Map" för att konfigurera kartrendering.

Skrolla neråt för att fortsätta

Fortsätt till "Main map".

Skrolla neråt för att fortsätta

Välj OSM Scout Server i listan med kartlager.

Skrolla neråt för att fortsätta

Alternativ

Efter valet av kartlager, sveper du tillbaka till verktygssidan. Välj "Options".

POI

I "Options", väljer du POI.

POI

Välj OSM Scout Server som leverantör för platssök och lokal sökning, som visat.

Navigation

Efter sökkonfigurationen sveper du tillbaka till "Options". I Options, väljer du "Navigation".

Navigation

I inställningarna för navigation, väljer du OSM Scout Server som ruttberäkningsleverantör.

Skrolla neråt för att fortsätta

Åtgärdsläge

Du är klar! Glöm inte att du måste ha OSM Scout Server och modRana igång samtidigt, för offline-åtgärder.
