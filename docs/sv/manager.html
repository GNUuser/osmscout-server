Landsinformation

För varje land eller område, visas också storlek och tidsinformation per datauppsättning.

Prenumerera

Tryck på Prenumerera för alla länder som du är intresserad av. Efter det, sveper du åt vänster för att komma tillbaka till karthanterarens huvudsida.

Skrolla neråt för att fortsätta

Alla prenumererade länder visas som på skärmklippet.

Skrolla neråt för att fortsätta

Ladda ner

Tryck på Ladda ner för att starta nerladdningen. För bästa resultat, bör du ansluta enhetens laddkabel och använda WiFi-nätverket.

Skrolla neråt för att fortsätta

När nerladdningarna är klara, kommer varje nerladdat land att ha en motsvarande ikon, som det visas på skärmklippet as shown on the screenshot. Du kan avsluta och återvända till karthanteraren när du vill. Notera att om nerladdningen misslyckas, kan du stänga felmeddelandet och försöka igen.
