Innan lagringsplatsen är allokerad och inställd, kommer du att få en uppmaning om att göra detta. Denna handledning visar hur du skapar lagringsplats på SD-kortet eller annan icke standardplats.

Skrolla neråt för att fortsätta

Starta FileCase

Starta en filhanterare på din Sailfish-enhet. Du kan t.ex öppna FileCase och svepa åt höger, som det visas på skärmklippet.

Skrolla neråt för att fortsätta

SD-kort

Tryck på SD-kort.

Skrolla neråt för att fortsätta

Skapa ny mapp

Gå till den plats på SD-kortet, där du vill ha din lagringsmapp och skapa den via toppmenyn, som det visas på skärmklippet.

Skrolla neråt för att fortsätta

Namnge mappen

Namnge mappen som du önskar. I exemplet heter den Maps.OSM.

Skrolla neråt för att fortsätta

Tillbaka till OSM Scout Server

Gå tillbaka till OSM Scout Server och öppna inställningarna från toppmenyn, som visat.

Skrolla neråt för att fortsätta

Välja lagringsplats

Öppna lagringsplatsdialogen som visas markerat på skärmklippet.

Skrolla neråt för att fortsätta

Navigera till den nyskapade mappen

Navigera till platsen för den nyskapade mappen. I detta exempel, börjar vi från /home/nemo och måste ta oss till /media/sdcard/, så navigeringen börjar med att flytta upp två nivåer genom att trycka som skärmklippet visar.

Skrolla neråt för att fortsätta

Gå till "media" när du är i root-mappen (/).

Skrolla neråt för att fortsätta

I media-mappen fortsätter du till din nyskapade mapp.

Skrolla neråt för att fortsätta

Välj lagringsmapp

Långtryck på den nya lagringsmappen tills kontextmenyn öppnas, tryck sedan "Välj" för att bekräfta ditt val.

Skrolla neråt för att fortsätta

Den nya lagringsmappen är vald, som visas på skärmklippet, under "Kartlagring". Fortsätt genom att acceptera de nya inställningarna.
