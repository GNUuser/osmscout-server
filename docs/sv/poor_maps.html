Starta Poor Maps

Starta Poor Maps och gå till verktygen genom att trycka på knappen i botten på skärmen (markerat med röd fyrkant på skärmklippet).

Skrolla neråt för att fortsätta

Baskarta

Tryck på "Baskartor och överlägg", för att konfigurera kartvisning.

Skrolla neråt för att fortsätta

Välj OSM Scout Server i listan över baskartor. Användning av @2x rekommenderas för snabbare rendering.

Skrolla neråt för att fortsätta

Sök

Välj "Sök", i Poor Maps verktygsmeny.

Välj OSM Scout Server som sökmotorleverantör.

Navigation

Välj "Navigation" i Poor Maps verkygsmeny, som visat.

Skrolla neråt för att fortsätta

Välj OSM Scout Server som sökmotorleverantör, i navigationsmenyn.

Skrolla neråt för att fortsätta

Driftläge

Du är klar! Glöm inte att OSM Scout Server och Poor Maps måste köras samtidigt för offline-användning.
