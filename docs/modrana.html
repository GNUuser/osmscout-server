Options

After selecting the map layer, swipe back to get to the main tools page. Select "Options".

POI

In Options, select POI.

POI

Select OSM Scout Server as a provider for place search and local search, as shown.

Navigation

After search configuration, swipe back to Options. In Options, select Navigation.

Navigation

In Navigation settings, select OSM Scout Server as a routing provider, as shown.
